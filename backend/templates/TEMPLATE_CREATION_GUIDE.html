üöÄ Testing Your Template

  1. Save your template as backend/templates/letterhead.docx
  2. Restart your backend server
  3. Go to your app and create a new MOM
  4. Use the new "Generate PDF from Template" option
  5. Download and check the PDF

üí° Tips for Best Results

  • Keep layouts simple: Complex designs may not convert perfectly
  • Use standard fonts: Arial, Calibri, Times New Roman work best
  • Test frequently: Generate PDFs while designing to catch issues early
  • Use tables: Word tables help maintain consistent layout
  • Avoid text boxes: Use regular text instead of floating text boxes
  • Set margins: Leave enough white space (at least 0.5" on all sides)

üîß PDF Conversion (Optional but Recommended)

For best PDF quality, install LibreOffice:

  1. Download from: https://www.libreoffice.org/download/
  2. Install to default location
  3. Restart your backend server

Without LibreOffice, the system will still work but may save files as DOCX instead of PDF.
